# prefer-inline-type-import

Prefer inline type import.

## Invalid

<!-- eslint-skip -->
```ts invalid
import type { Foo } from 'foo'; 
```

## Valid

<!-- eslint-skip -->
```ts valid
import { type Foo } from 'foo';

import type Foo from 'foo';

import type * as Foo from 'foo';

import type {} from 'foo';
```
